CoreModules.WEASEL.DisplayImageCommon¶

This module contains helper functions used by functions in modules DisplayImageColour.py & DisplayImageROI.py

closeSubWindow(self, objectName)[source]¶

Closes a particular sub window in the MDI

self - an object reference to the WEASEL interface. objectName - object name of the subwindow to be closed

displayColourTableInComboBox(cmbColours, colourTable)[source]¶

This function causes the combobox widget cmbColours to display the name of the colour table stored in the string variable colourTable.

cmbColours - name of the dropdown lists of colour map names colourTable - String variable containing the name of a colour table

getDICOMFileData(self)[source]¶

When a DICOM image is selected in the tree view, this function returns its description in the form - study number: series number: image name

Input Parameters

self - an object reference to the WEASEL interface.

fullImageName - string containing the full description of a DICOM image

getHistogramLevels(imv, spinBoxIntensity, spinBoxContrast)[source]¶

This function determines contrast and intensity from the image and set the contrast & intensity spinboxes to these values.

imv - pyqtGraph imageView widget spinBoxIntensity - name of the spinbox widget that displays/sets image intensity. spinBoxContrast - name of the spinbox widget that displays/sets image contrast.

getPixelValue(pos, imv, pixelArray, lblPixelValue)[source]¶

This function checks that the mouse pointer is over the image and when it is, it determines the value of the pixel under the mouse pointer and displays this in the label lblPixelValue.

pos - X,Y coordinates of the mouse pointer imv - pyqtGraph imageView widget pixelArray - pixel array to be displayed in imv lblPixelValue - Label widget that displays the value of the pixel under the mouse pointer

and the X,Y coordinates of the mouse pointer.

readLevelsFromDICOMImage(self, pixelArray)[source]¶

Reads levels directly from the DICOM image

self - an object reference to the WEASEL interface. pixelArray - pixel array to be displayed

centre - Image intensity width - Image contrast maximumValue - Maximum pixel value in the image minimumValue - Minimum pixel value in the image

setPgColourMap(colourTable, imv, cmbColours=None, lut=None)[source]¶

This function converts a matplotlib colour map into a colour map that can be used by the pyqtGraph imageView widget.

colourTable - name of the colour map imv - name of the imageView widget cmbColours - name of the dropdown lists of colour map names lut - name of the look up table containing raw colour data

setUpImageViewerSubWindow(self)[source]¶

This function creates a subwindow with a vertical layout & a missing image label.

self - an object reference to the WEASEL interface.

layout - PyQt5 QVBoxLayout vertical layout box lblImageMissing - Label displaying the text ‘Missing Image’. Hidden

until WEASEL tries to display a missing image

subWindow - An QMdiSubWindow subwindow

setUpViewBoxForImage(layout, spinBoxIntensity=None, spinBoxContrast=None)[source]¶

Sets up the pyqtGraph imageView widget and associated colour levels histogram for viewing a DICOM image.

layout - PyQt5 QVBoxLayout vertical layout box spinBoxIntensity - name of the spinbox widget that displays/sets image intensity. spinBoxContrast - name of the spinbox widget that displays/sets image contrast.
