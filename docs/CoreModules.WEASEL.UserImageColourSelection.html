CoreModules.WEASEL.UserImageColourSelection¶

This class module provides the class variables and functions needed to store and retrieve the data associated with changing the colour table, intensity and contrast levels in individual images in a DICOM series of images.

class UserSelection(listImageLists)[source]¶

Bases: object

clearUserSelection()[source]¶

Resets the colour table name, intensity and contrast levels to their default values. Also, sets applyUserSelectionToAnImage to False to show that the user has not selected a new colour table etc for one or more images in the series

getImageUpdateStatus()[source]¶
getSeriesUpdateStatus()[source]¶
returnImageNumber(imageName)[source]¶

Returns the ordinal number of the selected image in the list of lists, userSelectionList, that stores the user’s selected colour table, contrast level and intensity level for each image.

returnUserSelection(imageNumber)[source]¶

Returns the colour table name, intensity and contrast values selected by the user for the image that has ordinal position imageNumber in userSelectionList, a list of lists where each sublist represents an image in the series being viewed.
