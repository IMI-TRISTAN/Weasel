class Affine(affine=None, *, direction=None, spacing=None, origin=None)[source]¶

Bases: numpy.ndarray

Class for general (d+1)x(d+1) affine matrices, and in particular d=3 (3d space) Usage examples: >>> affine1 = Affine(np.eye(4)) >>> affine2 = Affine(direction=np.eye(3), spacing=[0.33, 1, 0.33], origin=[-90.3, 10, 1.44]) >>> index = [4, 0, 9] >>> coord = affine2.index2coord(index) >>> print(coord) [-88.98 10. 4.41]
